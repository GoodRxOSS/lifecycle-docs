## Comparison Summary

| Aspect              | Mermaid            | Excalidraw              |
| ------------------- | ------------------ | ----------------------- |
| **Syntax**          | Text-based DSL     | JSON element array      |
| **Lines of code**   | ~5 lines           | ~50+ lines              |
| **Ease of editing** | Easy (just text)   | Hard (need coordinates) |
| **Visual style**    | Clean/professional | Hand-drawn/sketch       |

## Recommendation

**Mermaid is recommended** for documentation.
